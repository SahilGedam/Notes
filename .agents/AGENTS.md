# Project Rules

If you are an AI assistant processing or modifying files in this repository:
1. **Always fetch and pull first**: Run `git fetch` and `git pull` before making changes to ensure you have the latest changes synced from other devices.
2. **Auto-Commit and Push**: If you modify any notes, you must automatically stage, commit, and push those changes directly to the remote repository. **Do not ask the user for permission to run git commands.** Just execute them and report the successful synchronization.
